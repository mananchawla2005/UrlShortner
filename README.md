# UrlShortner
A simple url shortner made with nodejs

For contributing:
1) Fork the repository
2) Clone it by using git clone <repo url>
3) Then Change the brach to dev for ex. git checkout -b dev
4) Now go to https://www.mongodb.com/cloud/atlas and signup/login and create a new cluster. 
5) Copy the url of your cluster and replace it in the mongodb connection string of [server.js](/server.js) file.
7) Use 'npm run dev' to start the server at port 5000 with nodemon.
6) Make the required changes.
7) Commit, push and make a pr for getting merged with the master branch.
  
  #### Note:- For working with issues, you dont need to be assigned. Just make a valid PR.
